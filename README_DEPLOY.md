# Bobek Power Solutions website package

This folder contains the publish-ready static website for `bobekpowersolutions.com`.

## Upload these files
Upload the **contents** of this folder to the web host root so that `index.html` sits at:

`https://bobekpowersolutions.com/index.html`

Do not upload only the ZIP itself unless your host automatically unzips it.

## Included

- `index.html` - homepage
- `404.html` - fallback page
- `assets/css/styles.css` - site styling
- `assets/js/main.js` - mobile menu and mailto quote helper
- `assets/images/` - local optimized images
- `assets/brand/bobek-logo.svg` - logo used in the header/footer
- favicon files: `favicon.ico`, `favicon.svg`, PNG sizes, Apple and Android icons
- `site.webmanifest`
- `robots.txt`
- `sitemap.xml`
- `CNAME` for GitHub Pages
- `_headers`, `_redirects`, and `netlify.toml` for Netlify/Cloudflare-style static hosting
